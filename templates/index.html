Data Visualisation and Image Transformation

This webpage performs 2 functions based on the input file.

  • If the uploaded file is a csv file then a pie and bar chart will be created taking 1st column as category and 2nd column as value
  • If the uploaded file is an image file[.png/.jpg/.jpeg] then it will be converted to Grayscale and an option to download the file will be available
